#  Movie Recommendation Web App-Page ( using grok API) 

A full-stack  web page using grok ai movie recommendation page.

## Tech Stack
- Frontend: React + Vite (Tailwind-CSS)
- Backend: Node.js + Fastify
- AI: Groq (LLaMA 3.1 – free)
- Database: SQLite

## Features
- User inputs movie preference
- AI returns 5(as asked) movie recommendations
- Recommendations stored in SQLite database
- Fully deployed frontend and backend

## Project Structure
- /frontend → React application
- /backend → API server
